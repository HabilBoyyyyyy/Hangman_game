# Audio Assets Folder

Place your downloaded `.mp3` or `.wav` files here. Ensure the filenames match the ones specified below so the code can load them automatically.

## Required Files:

1. `bgm-noir.mp3` - Background music (Looping jazz or rain).
2. `sfx-key.mp3` - Single typewriter key click (UI navigation & typing).
3. `sfx-correct.mp3` - Typewriter bell ding (Correct guess).
4. `sfx-wrong.mp3` - Heavy rubber stamp thud (Wrong guess).
5. `sfx-duplicate.mp3` - Paper scratch / rustle (Already guessed letter).
6. `sfx-win.mp3` - Rapid typing + final stamp (Game won).
7. `sfx-lose.mp3` - Metal jail cell door slam (Game lost).
8. `sfx-page.mp3` - Paper rustle/page flip (Menu transitions).
